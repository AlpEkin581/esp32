## 🌍 English 

This project allows you to draw on an OLED screen connected to an **ESP32** via **Bluetooth** using a web-based interface.

### 🚀 Features
* 📱 **Mobile Friendly:** Supports touch gestures for drawing on smartphones.
* 📶 **Wireless:** Real-time data transmission over Bluetooth.
* 🎨 **Color Selection:** Pick any color and send it to the display instantly.

### 🛠️ Hardware Requirements
* ESP32 Development Board
* 128x64 OLED Display (SPI)
* Jumper Wires & Breadboard

---

## 🇹🇷 Türkçe 

Bu proje, bir web arayüzü üzerinden **Bluetooth** kullanarak **ESP32**'ye bağlı bir OLED ekrana çizim yapmanızı sağlar.

### 🚀 Özellikler
* 📱 **Mobil Uyumlu:** Akıllı telefonlarda dokunmatik çizim desteği.
* 📶 **Kablosuz:** Bluetooth üzerinden anlık veri iletimi.
* 🎨 **Renk Seçimi:** İstediğiniz rengi seçip ekrana gönderebilirsiniz.

### 💻 Nasıl Kullanılır? / How to Use?
1. Upload the code to your ESP32 / Kodu ESP32'ye yükleyin.
2. Visit the [Web Site](https://AlpEkin581.github.io/esp32/).
3. Connect and start drawing! / Bağlanın ve çizmeye başlayın!
